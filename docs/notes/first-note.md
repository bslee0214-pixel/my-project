# 바이브 코딩 메모

> 오늘 배운 바이브 코딩(Vibe Coding)에 대한 간단한 기록입니다.

---

## 소개

오늘은 AI와 함께 개발하는 방식인
"바이브 코딩"에 대해 공부했다.

기존처럼 모든 코드를 직접 작성하는 방식이 아니라,
아이디어와 의도를 설명하면서 AI와 협업하는 형태가 인상적이었다.

---

## 목차

1. 바이브 코딩이란?
2. 느낀 점
3. 장점
4. 앞으로 해볼 것

---

## 1. 바이브 코딩이란?

바이브 코딩은 AI에게 다음과 같은 내용을 설명하며 개발하는 방식이다.

- 원하는 기능
- 화면 구조
- 동작 방식
- 코드 수정 방향

AI는 이를 바탕으로 코드 초안을 생성해준다.

---

## 2. 느낀 점

생산성이 매우 빠르다는 점이 가장 인상적이었다.

특히 반복적인 코드 작성이나
기본 구조 생성 속도가 굉장히 빨랐다.

다만 완전 자동이라기보다는
"AI와 함께 개발한다"는 느낌에 가까웠다.

---

## 3. 장점

- 빠른 프로토타이핑
- 반복 작업 감소
- 아이디어 구현 속도 향상
- 새로운 기술 학습 보조

---

## 4. 앞으로 해볼 것

앞으로 아래와 같은 작업에 활용해볼 계획이다.

- WinUI 앱 개발
- 자동화 도구 제작
- 게임 보조 앱 실험
- 문서 자동 생성

---

## 마무리

AI 시대에는 코드를 많이 아는 것도 중요하지만,
원하는 결과를 잘 설명하는 능력도 점점 중요해지는 것 같다.
